Building HTML Lists

Embedded Style Sheet - An embedded style sheet is a set of Cascading Style Sheets (CSS) rules written directly within an HTML document's head section using style tags, appyling styles specifically to that single document rather than the whole website.

External Style Sheet - An external style sheet is a separate file containing Cascading Style Sheets (CSS) rules, typically with a .css extention, that is linked to one or more HTML document to control their visual appearence.

Inline Style - An inline style in HTML refers to applying CSS styles directly to a single HTML element using the style attribute within that element's opening tag.
